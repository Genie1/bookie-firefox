Congrats, you just installed the Bookie extension!

Before you can start using it,

  1. Set the Bookie extension preferences found under Tools > Add-Ons
  2. Click the "Preferences" button next to the Bookie Add-On
  3. Fill in your username and API key
  4. Start bookmarking!

The API key can be found in your user info section on the Bookie site.
